Installation

Place ezlib.lua in your project repository. Require it in local scripts in Roblox Studio using require.

-- inside a LocalScript

local ezlib = require(game.ReplicatedStorage:WaitForChild("ezlib"))

Quickstart

Minimal example that creates a window, adds a button and an inputbox, and shows a notification.

local ezlib = require(game.ReplicatedStorage:WaitForChild("ezlib"))

local lib = setmetatable({}, ezlib) local win = lib:Window("Demo Window", UDim2.new(0,420,0,360)) local content = Instance.new("Frame") content.Size = UDim2.new(1,0,1,-40) content.Position = UDim2.new(0,0,0,40) content.Parent = win

local btn = lib:Button(content, "Click me", function() lib:Notification("Button clicked", 2) end)

local box = lib:Inputbox(content, "Type name...", function(text) lib:Notification("You typed: "..text, 2) end)

lib:SetTheme({ BackgroundColor = Color3.fromRGB(32,32,32), TextColor = Color3.fromRGB(240,240,240), AccentColor = Color3.fromRGB(255,120,80), OutlineColor = Color3.fromRGB(60,60,60) })

API Reference

All functions are defined on the ezlib table.

ezlib:SetTheme(themeTable)

Applies a theme object to existing UI elements that ezlib created. Theme keys used by ezlib:

KeyDescription
BackgroundColorFrame background color
TextColorTextLabel/TextButton/TextBox color
AccentColorMain accent (buttons, fills)
OutlineColorUIStroke color

ezlib:Window(title, size, position)

Creates a Window (custom Frame) and sets it as self.MainUI. Returns the Window instance.

local win = lib:Window("Title", UDim2.new(0,400,0,300), UDim2.new(0.5,-200,0.5,-150))

ezlib:Button(parent, text, callback)

Creates a Button inside parent. The callback runs on click.

ezlib:Inputbox(parent, placeholder, callback)

Creates an Inputbox. When user presses Enter, callback(text) is called.

ezlib:Checkbox(parent, label, default, callback)

Creates a Checkbox with boolean state changes sent to callback.

ezlib:Slider(parent, min, max, default, callback)

Creates a horizontal Slider. Callback receives numeric value while dragging.

ezlib:Toggle(parent, label, default, callback)

Creates a Toggle switch. Callback receives boolean state when toggled.

ezlib:Notification(text, duration)

Creates a temporary notification attached to self.MainUI.

Components & Styling

ezlib uses these custom class names in place of default Roblox types when describing components:

  • Window — top-level container (Frame)
  • Button — clickable control (TextButton)
  • Inputbox — TextBox
  • Checkbox — checkbox control composed of Frame/TextButton/TextLabel
  • Slider — slider composed of Frame(s)
  • Toggle — on/off control
  • Notification — temporary TextLabel popup

Theme System

Change colors at runtime by passing a table to SetTheme. Example themes:

lib:SetTheme({

BackgroundColor = Color3.fromRGB(18,18,18), TextColor = Color3.fromRGB(230,230,230), AccentColor = Color3.fromRGB(0,170,255), OutlineColor = Color3.fromRGB(50,50,50) })

Examples

Full example: Settings panel

local ezlib = require(game.ReplicatedStorage:WaitForChild("ezlib"))

local lib = setmetatable({}, ezlib) local win = lib:Window("Settings", UDim2.new(0,420,0,420)) local content = Instance.new("Frame") content.Size = UDim2.new(1,0,1,-40) content.Position = UDim2.new(0,0,0,40) content.Parent = win

local soundToggle = lib:Toggle(content, "Sound", true, function(state) print("sound state", state) end) local volSlider = lib:Slider(content, 0, 100, 50, function(value) print("volume", value) end) local saveBtn = lib:Button(content, "Save", function() lib:Notification("Settings saved", 2) end)

FAQ

Q: Where do I put ezlib.lua?
A: Recommended: ReplicatedStorage/ezlib.lua and require from LocalScripts.

Q: Can I extend components?
A: Yes — ezlib returns raw Roblox Instances so you can add children or change properties after creation.
